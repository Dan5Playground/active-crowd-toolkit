Detailed Description

Generated algorithm for performing inference.

If you wish to use this class directly, you must perform the following steps: 1) Create an instance of the class. 2) Set the value of any externally-set fields e.g. data, priors. 3) Call the Execute(numberOfIterations) method. 4) Use the XXXMarginal() methods to retrieve posterior marginals for different variables.
